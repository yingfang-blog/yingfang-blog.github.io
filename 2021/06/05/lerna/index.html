本地调试方法
Event Loop

在JavaScript中，任务被分为两种，一种宏任务（MacroTask），一种微任务（MicroTask）。

  • 宏任务

    script全部代码、setTimeout、setInterval、setImmediate（浏览器暂时不支持，详情可见MDN）、I/O、UI Rendering

  • 微任务

    Process.nextTick(Node独有)、Promise、Object.observe(废弃)、MutationObserver

JavaScript

  • 执行宏任务
  • 宏任务里面的放入执行栈（后进先出）中，依次执行，遇到微任务放入微任务队列，宏任务放入宏任务队列
  • 查看微任务队列，如果有任务，就采用先进先出的方式，将微任务放入执行栈中依次执行，直到微任务队列清空
  • 查看宏任务队列，循环到1

Node

  • 当每个阶段完成后，如果存在nextTick队列，就会清空队列中的所有回调函数，并且优先于其他微任务（Micro）执行
